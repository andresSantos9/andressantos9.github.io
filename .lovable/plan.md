

# Academic Web Page

A classic, formal academic webpage — single scrolling page with sticky top navigation, profile photo, and serif-accented typography.

## Design
- **Style**: Classic academic aesthetic — serif headings (Georgia/Times), clean layout, subtle borders and dividers, muted color palette (navy, ivory, dark grays)
- **Navigation**: Sticky top navbar with links to each section (About, Employment, Research, Teaching, Contact)
- **Profile photo**: Circular placeholder image in the About/hero section

## Sections

1. **Hero / About Me** — Name, title/affiliation, profile photo placeholder, short bio paragraph

2. **Employment History** — Timeline-style list of positions (institution, role, dates) with classic bordered layout

3. **Research** — Research interests/areas, plus publications subsection with:
   - Each paper displayed as a card with:
     - Thumbnail preview (image or video thumbnail with play icon overlay)
     - Title, authors, venue, year
     - Link to video presentation (if available)
     - Download/view paper link
   - Grid or list layout for papers

4. **Teaching** — Table or list of courses (name, code, semester, institution)

5. **Contact** — Email, office, academic profile links (Scholar, ORCID), simple contact form

## Footer
- Affiliation, copyright

All content uses placeholder data for easy replacement.
